eval :: (ArrowChoice c, ArrowFix Expr v c, ArrowEnv Text v env c, ArrowFail e c, IsString e, IsNum v c, IsClosure v env c, Join c ((v, Text), Text) v, Join c (Expr, Expr) v) => c Expr v Source #

The Generic interpreter for PCF. It uses the IsNum and IsClosure interfaces to operate on numeric and closure values. Furthermore, it uses ArrowFix, ArrowEnv, and ArrowFail to compute the fixpoint, operate on the environment, and fail with an error message.
