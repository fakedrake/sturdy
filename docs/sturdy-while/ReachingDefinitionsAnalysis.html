run :: (?bound :: IV) => Int -> [LStatement] -> [(Label, Map Text (Pow Label))] Source #

Calculates the entry sets of which definitions may be reached for each statment. The analysis instantiates the generic interpreter run with analysis components for fixpoint computation (FixT), termination (TerminatingT), failure (ErrorT), store (StoreT), environments (EnvT), and values (IntervalT)
