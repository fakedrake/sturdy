type Widening a = a -> a -> (Stable, a) Source #

▽ has to be an upper bound operator, i.e. x ⊑ x ▽ y and y ⊑ x ▽ y. Furthermore, iterating ▽ on an ascending chain has to stabilize: Let x1, x2, ... xn be an infinite ascending chain, then x1, x1 ▽ x2, (x1 ▽ x2) ▽ x3, ... (similar to left fold) has a limit:

... finite ▽ / ▽ / ▽ / x1 ⊑ x2 ⊑ x3 ⊑ x4 ⊑ ... infinite

Furthermore, we allow our widening operators to maintain some state between each iteration.
