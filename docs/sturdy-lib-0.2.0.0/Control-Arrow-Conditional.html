if_ :: Join c (x, y) z => c x z -> c y z -> c (v, (x, y)) z Source #

if_ f g -< (v,(x,y)) performs a case distinction on the given value v and executes either (f -< x) or (g -< y). Abstract instances might join the results of f and g.
