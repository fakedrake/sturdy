type StackWidening stack a = a -> stack a -> (a, stack a) Source #

A stack widening operator (▽ :: s -> a -> (s,a)) follows the same idea as a regular widening operator, but does not have the restriction that the input chain has to be ascending: Given an infinite series x1, x2, x3, x4 ..., the stack widening operator produces a series x1 ▽ s1 = (x1',s2) & x1 ⊑ x1' x2 ▽ s2 = (x2',s3) & x2 ⊑ x2' x3 ▽ s3 = (x3',s4) & x3 ⊑ x3' x4 ▽ s4 = (x4',s5) & x4 ⊑ x4' ... such that x1', x2', x3' ... repeats itself, i.e. there exists n,m with n /= m and xn' = xm'.
